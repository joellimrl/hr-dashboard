# Backend for HR Dashboard

Middleware microservice built on NodeJS and ExpressJS

Utilises MongoDB and RSA Key encryption for authentication

## Getting Started

Install dependencies (after cloning project)

```
npm install
npm run start
```

### Prerequisites

MongoDB
- DB name: hr-dashboard
- Collections:
    - authentication
    - employee-details
    - employee-list
    - expo-push-tokens
